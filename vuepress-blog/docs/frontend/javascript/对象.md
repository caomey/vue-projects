# 对象

对象有属性和方法

```js
var obj = {
  name: "xiaowei",
  age: "27",
  drink: function () {
    console.log("i am drinking");
  },
  eat: function (food) {
    console.log("i am eating" + food);
  },
};

// 增删改查
console.log(obj.name); //查询
obj.age = 28;// 改
obj.sex = 'male'; // 增加属性
delete obj.age; // 删除属性
```

## 构造函数

和函数的区别，构造函数，遵循大驼峰命名规则

```js
function Car(color) {
  this.name = "BMW";
  this.color = color;
  this.height = "1400";
  this.lang = "4900";
  this.weight = 1000;
  this.heelth = 100;
  this.run = function(){
        this.health --;
    }
}

var car = new Car('red');
var car1 = new Car();
console.log(car);
```

## 构造函数内部原理

new之后

1. 在函数体最前面隐式的加上var  this = {}
2. 执行this.xxx = xxx;
3. 隐式的return this

模拟构造函数this

```js
function Person(name, height) {
  var that = {};
  that.name = name;
  that.height = height;
  return that;
}
var person = Person(" xiaowang", 180);
var person1 = Person(" xiaozhang", 175);
console.log(person.name);
```
